# ScoutsLens - 篮球数据分析与阵容优化系统

## 项目简介

Scout's Lens是一个篮球经理游戏，通过预测 NBA 球员的表现和统计数据，选出球队阵容。


## 游戏规则

### 基本规则

Scout's Lens 是一个基于真实 NBA 数据的篮球经理游戏，玩家需要根据球员的历史表现和统计数据来组建最优阵容。

### 阵容构成

#### 首发阵容 (5人)
- **控球后卫 (PG)** - 1人
- **得分后卫 (SG)** - 1人
- **小前锋 (SF)** - 1人
- **大前锋 (PF)** - 1人
- **中锋 (C)** - 1人

#### 替补阵容 (7人)
- 替补球员可以是任何位置的球员，共7人

### 薪资限制

- **总薪资上限**: 187,895,000 美元（2025-26赛季 NBA 奢侈税线）
- 首发阵容和替补阵容所有球员的薪资总和不得超过此限制

### 球员位置兼容性

球员可以根据其官方位置担任多个位置：

| 球员位置 | 可担任位置 |
|---------|-----------|
| Guard   | PG, SG    |
| Guard-Forward | SG, SF |
| Forward-Guard | SG, SF |
| Forward | SF, PF    |
| Forward-Center | PF, C  |
| Center-Forward | PF, C  |



#### 评分公式

```
评分 = (
    三分球命中数 × 1.5
    + 两分球命中数 × 1
    + 罚球命中数 × 0.5
    + 进攻篮板 × 1
    + 防守篮板 × 0.7
    + 助攻 × 1
    + 抢断 × 1.2
    + 盖帽 × 1.2
    - 投篮未命中次数 × 0.7
    - 罚球未命中次数 × 0.4
    - 失误 × 1.2
    - 个人犯规 × 0.4
)

如果上场时间 > 0：
    如果球队获胜，评分 += 2
    如果球队失败，评分 -= 2
```

### 阵容评分

阵容评分是球员评分的加权求和，首发球员系数为2，替补球员系数为1。
