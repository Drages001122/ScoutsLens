# Scout's Lens - NBA 球员阵容管理工具

## 项目简介

Scout's Lens 是一个基于 Streamlit 的 NBA 球员阵容管理工具，允许用户：

- 浏览和筛选 NBA 球员
- 构建符合薪资上限的阵容
- 管理首发和替补球员
- 导出阵容为 CSV 文件

## 快速开始

### 方法一：一键启动（推荐）

1. 确保你的电脑上安装了 Python 3.8 或更高版本
2. 下载本项目到本地
3. 双击运行 `start_server.bat` 文件
4. 批处理文件会自动：
   - 检查虚拟环境是否存在（如果不存在则创建）
   - 激活虚拟环境
   - 启动 Streamlit 服务器（虚拟环境已包含所有依赖）
5. 服务器启动后，会自动在浏览器中打开应用

### 方法二：手动启动

1. 确保你的电脑上安装了 Python 3.8 或更高版本
2. 下载本项目到本地
3. 创建并激活虚拟环境：
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
4. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
5. 启动 Streamlit 服务器：
   ```bash
   streamlit run app.py
   ```

## 项目结构

- `app.py` - 主应用文件
- `start_server.bat` - 一键启动脚本
- `requirements.txt` - 依赖包列表
- `球员信息.csv` - 球员数据文件（需自行准备）

## 注意事项

1. **球员数据文件**：项目需要 `球员信息.csv` 文件来加载球员数据，请确保该文件存在于项目根目录
2. **网络连接**：首次运行时需要下载依赖包，因此需要网络连接
3. **端口占用**：默认情况下，Streamlit 会使用 8501 端口，如果该端口被占用，可能会自动使用其他端口

## 技术栈

- Python 3.8+
- Streamlit
- Pandas
- NumPy

## 功能特点

- **球员筛选**：支持按位置、球队、薪资范围筛选球员
- **阵容管理**：可视化管理首发和替补阵容
- **薪资计算**：自动计算总薪资并检查是否符合薪资上限
- **位置检查**：确保首发阵容符合 2-2-1（2 后卫、2 前锋、1 中锋）的位置要求
- **数据导出**：支持将阵容导出为 CSV 文件

## 问题排查

1. **依赖安装失败**：请确保你的网络连接正常，或者尝试使用国内镜像源
2. **数据加载失败**：请检查 `球员信息.csv` 文件是否存在且格式正确
3. **端口占用**：如果 8501 端口被占用，可以尝试在启动时指定其他端口：
   ```bash
   streamlit run app.py --server.port 8502
   ```
