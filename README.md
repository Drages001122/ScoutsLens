# Scout's Lens - NBA 球员阵容管理工具

## 游戏规则

### 1. 阵容构建规则

- **阵容规模**：每支球队最多可以拥有 15 名球员
- **首发阵容**：必须包含 5 名球员，且符合 2-2-1 位置要求（2 后卫、2 前锋、1 中锋）
- **替补阵容**：最多可以拥有 10 名球员，位置分布无限制

### 2. 薪资上限规则

- **总薪资限制**：默认薪资上限为 1.32 亿美元（2023-24 赛季 NBA 薪资帽）
- **薪资计算**：系统会自动计算所有球员的总薪资
- **薪资检查**：当总薪资超过薪资上限时，系统会发出警告
- **薪资调整**：用户可以通过增加/移除球员来调整总薪资，使其符合上限要求

### 3. 球员选择规则

- **球员筛选**：可以通过以下条件筛选球员：
  - 位置（后卫、前锋、中锋）
  - 球队
  - 薪资范围
- **球员状态**：球员可以处于以下状态：
  - 自由球员池（可选择）
  - 阵容中（已选择）
  - 首发阵容（已安排首发）

### 4. 阵容管理规则

- **首发调整**：用户可以将阵容中的球员拖拽到首发阵容，或从首发阵容拖拽回替补阵容
- **位置要求**：首发阵容必须满足 2-2-1 位置配置，否则系统会提示错误
- **球员移除**：可以从阵容中移除球员，使其回到自由球员池

### 5. 数据导出规则

- **导出格式**：支持将当前阵容导出为 CSV 文件
- **导出内容**：导出文件包含球员姓名、位置、球队、薪资等基本信息
- **导出路径**：导出文件会保存到用户指定的位置

## 球员评分系统

### 核心公式（简化版）
评分计算公式为：

```
SCORE=3P+(2P×0.8)+(FT×0.5)+ORB+(DRB×0.7)+AST+STL+BLK−(FGA−FG)×0.7−(FTA−FT)×0.4−TOV−PF×0.4
```

### 参数说明

| 符号 | 含义 |
|------|------|
| MP   | 上场时间 |
| 3P/2P/FT | 三分命中数 / 两分命中数 / 罚球命中数 |
| FGA/FG | 投篮出手数 / 投篮命中数 |
| FTA/FT | 罚球出手数 / 罚球命中数 |
| ORB/DRB | 进攻篮板 / 防守篮板 |
| AST/STL/BLK | 助攻 / 抢断 / 盖帽 |
| TOV/PF | 失误 / 犯规 |

### 球队获胜对评分的影响

- **获胜加成**：当球员所在球队获胜，且球员登场时，其评分将获得额外 2分 的加成；当球员所在球队落败，且球员登场时，其评分将扣除 2分。
